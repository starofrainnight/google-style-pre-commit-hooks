A series pre-commit hooks that support format source by google style

* License: Apache-2.0

Usage
---------

::

    repos:
    - repo: https://github.com/starofrainnight/google-style-pre-commit-hooks
      sha: b7e9e7fcba4a5aea463e72fe9964c14877bd8130
        hooks:
        - id: google-java-style
